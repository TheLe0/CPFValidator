# CPFValidator
CPF Validator using Sockets(TCP and UDP)

## Notas ##

- Deve ser executado o visual studio como administrador para poder rodar o programa;
- Para usar, deve ser rodado ao mesmo tempo duas vezes o programa(em duas instâncias do VS diferentes), uma para
o cliente outra para o servidor(sempre execute o servidor antes).
